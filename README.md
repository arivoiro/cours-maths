# 📚 Cours Maths — Plateforme de soutien scolaire en ligne

Un site vitrine moderne développé en **React** pour présenter des cours de mathématiques en ligne, les tarifs, la méthodologie et permettre aux élèves ou parents de contacter facilement l’enseignant.

🌐 **Démo en ligne** : [https://maths-par-allan.vercel.app](https://maths-par-allan.vercel.app)

---

## 🚀 Fonctionnalités

- Page d'accueil avec section héro et présentation
- Pages dédiées : Méthodologie, Tarifs, FAQ, Contact
- Formulaire de contact fonctionnel via [Formspree](https://formspree.io)
- Design responsive mobile & desktop
- Styles modernes avec **SCSS** et variables personnalisées
- Navigation fluide grâce à **React Router**

---

## 🛠️ Stack technique

- **React 18**  
- **SCSS** pour la mise en forme  
- **React Router** pour la navigation  
- **Formspree** pour la gestion des formulaires  
- Hébergement sur **Vercel**

---

## 🧰 Installation et développement local

- git clone https://github.com/arivoiro/cours-maths.git
- cd cours-maths
- npm install
- npm start

Le site sera accessible sur http://localhost:3000
